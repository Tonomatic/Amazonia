Backend

create backend/server.js

go into package.json, and add the line "type": "module",

node backend/server/js to start server

npm i --save-dev nodemon to facilitate development on the backend

go into package.json and delete test script line to

"start": "nodemon --watch backend --exec node --experimental-modules backend/server.js"

This line will watch for changes in the backend folder and exectue node, at backend/server
by using npm start
